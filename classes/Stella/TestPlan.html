requests [RW] An array of Stella::TestPlan::Request objects representing all “primary” requests for the given test plan (an html page for example). base_uri= (*args)

A string to be parsed by URI#parsed or a URI object. The host and port are added to +@servers+ in the form “host:port”. The protocol is stored in +@protocol+. NOTE: The protocol is used as a default for the test and if it’s already set, this method will not try to overwrite it.
