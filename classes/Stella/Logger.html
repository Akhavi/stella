["new\n", "(args={})\n"]

args is a hash of initialization arguments

  • :info_logger The IO class for info level logging. Default: STDOUT
  • :error_logger The IO class for error level logging. Default: STDERR
  • :debug_logger The IO class for error level logging. Default: STDERR
  • :debug_level An integer from 0 to 4 which determines the amount of debugging output. Default: 0.
